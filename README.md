# Trivia CLI Game

This project is a command-line trivia game built with JavaScript and Node.js.  
It runs entirely in the terminal and lets users answer multiple-choice trivia questions while being timed.

This lab was created to practice core JavaScript concepts including:
- functions and program flow
- arrays and objects
- user input through the CLI
- timers and asynchronous behavior

---

## How It Works

- The game starts by asking the user if they want to play
- Trivia questions are shown one at a time in the terminal
- Each question has multiple-choice answers
- The user receives immediate feedback after each answer
- There is a time limit per question and a total time limit for the entire game
- At the end, the final score is displayed along with a review of missed questions
-This game is very challenging 

---

## Technologies Used

- JavaScript (Node.js)
- `readline` / `readline/promises` for CLI input
- Built-in timers for managing game time

No external libraries are required.

---
